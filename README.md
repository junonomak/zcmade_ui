# cmade_ui

## 介绍

这是我学习vue.js过程中创建的一个基于vue3的ui库

## 开始使用

使用本框架前，请css中开启 border-box
IE8以上浏览器都支持该样式

1. 安装
```
    npm install cmade_ui -s
```
2. 引入cmade_ui组件
```
    在需要引入组件的.vue文件中按需引入组件
    import { cButton } from 'cmade_ui' 
```
3. 在main.js中引入css文件
```
    import './assets/iconfont/iconfont.css'
```
